# Matrix Portfolio

A single-page developer portfolio built with React, TypeScript, Vinext, and
Cloudflare Sites.

## Development

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Edit portfolio copy and project details in `lib/portfolio-content.ts`.

## Validation

```bash
npm run lint
npm test
```

The Sites project identity and deployment bindings live in
`.openai/hosting.json`.
